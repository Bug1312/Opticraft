Opticraft

Opticraft is a complete guide for an optimized 1.12.2 modded experience.

Client and Server Mods

  • CensoredASM

    CensoredASM is one of the most impactful mods in this guide, it aims at reducing the ram usage of minecraft making it faster and reducing the ram usage up to 4 times on big modpacks. It also prevent the game from crashing, and fixes bugs

    Require MixinBooter

  • FastWorkbench

    FastWorkbench aims to improves performance of all crafting-related functions, it achives this by fixing bugs and caching recipes. It also removes the recipe book which mean that recipes are not forcibly synced on login which improve loging times by a lot.

  • FoamFix

    FoamFix also aims at reducing the ram usage of minecraft and reducing tps usage by doing networking optimization and fixing bugs. It also does other more minor optimizations.

  • Phosphor

    Phosphor aims at optimizating the lighting part of minecraft by entirly replacing the lighting engine, it results in improved fps and improved world generation speed.

    Incompatible with Cubic Chunks

  • Sledgehammer

    Sledgehammer aims at smashing out the stupid out of the client and the server, by optimizing and cleaning up client and server code. It speeds up and improves connection to servers, reduces network usage. It also prevent crashes, fixes bugs and quality of life improvement.

  • Surge

    Surge aims at optimizing load time by cleaning up Forge code and turning off debuging code. It also improve performance with sheeps and fix a bug with the anvil renaming.

Client Only Mods

  • Entity Culling

    Entity Culling aims to optimize rendering of entity and tile entity by skipping the rendering of those that are hidden.

    Require MixinBooter Require RenderLib

  • OptiFine

    Optifine aims to optimize rendering by optimizing rendering code. It also add support for shaders hd textures and more.

  • Particle Culling

    Particle Culling aims to optimize rendering of particles by skipping the rendering of those that are hidden.

Server Only Mods (If you play singleplayer still install them)

  • AI Improvements

    AI Improvements aims at optimizing AI code, it achieves this by optimizing entity code while preserving default entity behavior.

  • Dynamic View

    Dynamic View aims to optimize server view distance by reducing it when the server lags and increasing it when the server has spare performance.

  • XP Orb Clump

    XP Orb Clump aims at reducing the lag from XP orbs by clumping them together. The main difference between this and Clumps is that this doesn't add a new entity.
