Opticraft

Opticraft is a complete guide for an optimized 1.12.2 modded experience.

If a mod is not here it has a reason, so don't install optimization mods that are not on here. If you think something is missing from Opticraft report it here.

V1.9

Client and Server Mods

  • CensoredASM

    CensoredASM is one of the most impactful mods in this guide, it aims at reducing the ram usage of Minecraft, making it faster and reducing the ram usage up to 4 times on big modpacks. It also prevents the game from crashing, and fixes bugs

    Require MixinBooter

  • FoamFix

    FoamFix also aims at reducing the ram usage of Minecraft and reducing tps usage by doing networking optimization and fixing bugs. It also does other more minor optimizations.

  • Hesperus

    Hesperus aims at optimizing the lighting part of Minecraft by entirely replacing the lighting engine, it results in improved fps and improved world generation speed. Hesperus is a fork of Phosphor which improves compatibility and fixes bugs.

    Incompatible with Cubic Chunks

  • Sledgehammer

    Sledgehammer aims at smashing out the stupid out of the client and the server, by optimizing and cleaning up client and server code. It speeds up and improves connection to servers, reduces network usage. It also prevents crashes, fixes bugs and quality of life improvement.

    Require MixinBooter

  • Universal Tweaks

    Universal Tweaks aims at superseding tweaking, bug fixing and small performance mods, it also improves performance in other ways.

    All of its tweaks can be turned on or off.

    Universal Tweaks optimizes load time by cleaning up and optimizing Forge code and turning off debugging of sounds, replacing Surge.

    It improves AI performance by optimizing entity code while preserving default entity behavior.

    Require MixinBooter

Client Only Mods

  • Entity Culling

    Entity Culling aims to optimize rendering of entity and tile entity by skipping the rendering of those that are hidden.

    Entity Culling only improves FPS when you have a lot of entities on screen, otherwise it decreases FPS so only install it when entities make you lag.

    Not included in Opticraft instance.

    Require MixinBooter Require RenderLib

  • Nothirium

    Nothirium aims to optimize rendering of chunks by rewriting the chunk rendering engine, which greatly increases performance.

    Require MixinBooter Require RenderLib Incompatible with Cubic Chunks Incompatible with Little Tiles Incompatible with Far Plane Two Causes artifacts with Multiblocked multiblocks

  • OptiFine

    Optifine aims to optimize rendering by optimizing rendering code. It also adds support for shaders and more.

  • Particle Culling

    Particle Culling aims to optimize rendering of particles by skipping the rendering of those that are hidden.

Server Only Mods (If you play singleplayer still install them)

  • Dynamic View

    Dynamic View aims to optimize server view distance by reducing it when the server lags and increasing it when the server has spare performance.

  • Multithreaded Noise

    Multithreaded Noise aims optimizing world generation by multithreading noise, which prevents noise generation from lagging the game and makes it faster.

  • XP Orb Clump

    XP Orb Clump aims at reducing the lag from XP orbs by clumping them together. The main difference between this and Clumps is that this doesn't add a new entity.

Bug Fixing Only Mods (For Server mods still install them in singleplayer)

  • DupeFix Project

    DupeFix Project aims at fixing dupes in Minecraft and mods. Never blacklist items because they cause a dupe again!

    Client & Server

  • McMouser

    McMouser aims at fixing bugs with mouse on Mac Minecraft.

    Not included in Opticraft instance.
